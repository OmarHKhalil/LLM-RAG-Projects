# LLM & RAG Projects

This repository contains practical examples of AI projects using LLM and RAG techniques.

## Projects Included

- AskMyDocs: A project for asking questions about PDF files using RAG.
- Genai-Finance-Chatbot: A finance FAQ chatbot using embeddings and Chroma.

## Quick Start

1. Open the folder for the project you want to run.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the notebook or application inside the project folder.

## Notes

- The goal of this repository is to showcase simple and practical LLM and RAG ideas.
- You can modify or expand these projects as needed.
